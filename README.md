# AI PDF Chatbot

An AI-powered PDF chatbot built using:

- Streamlit
- Gemini 2.5 Flash
- Sentence Transformers
- RAG (Retrieval Augmented Generation)

## Features

- Upload PDF
- Ask questions
- Semantic Search
- Gemini-powered Answers
- Chat History

## Tech Stack

- Python
- Streamlit
- PyMuPDF
- Sentence Transformers
- Google Gemini

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
